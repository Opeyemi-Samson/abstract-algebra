§ 1

Why Normal Subgroups?

In group theory, we can always form cosets of any subgroup H in a group G. Given H ≤ G, for each g ∈ G we get a left coset gH = {gh : h ∈ H} and a right coset Hg = {hg : h ∈ H}.

A natural question arises: can we make the set of all cosets into a group itself? We want to define multiplication on cosets as:

(aH)(bH) = (ab)H

But there's a problem. This operation must be well-defined — it cannot depend on which representatives a, b we pick for the cosets. If we pick a′ ∈ aH and b′ ∈ bH instead, we need (a′b′)H = (ab)H.

It turns out this works precisely when left and right cosets of H coincide — which is exactly the condition of being a normal subgroup. This is the deep reason why normal subgroups matter: they are exactly the subgroups that allow us to build quotient groups.

Intuition: A normal subgroup is one that "looks the same from both sides" — conjugating it by any group element gives back the same subgroup. It is a subgroup that the whole group "respects."
§ 2

The Definition

Definition — Normal Subgroup

A subgroup N of a group G is called a normal subgroup, written N ◁ G, if for every g ∈ G and every n ∈ N, the conjugate gng⁻¹ lies in N. That is:

N ◁ G ⟺ gNg⁻¹ ⊆ N for all g ∈ G where gNg⁻¹ = { gng⁻¹ : n ∈ N }

Since we also have g⁻¹Ng ⊆ N (apply the condition to g⁻¹), this actually gives equality:

N ◁ G ⟺ gNg⁻¹ = N for all g ∈ G

Notation

  • N ◁ G — N is a normal subgroup of G
  • N ◁◁ G — N is a proper normal subgroup (N ≠ G)
  • The map φ_g : n ↦ gng⁻¹ is called conjugation by g
§ 3

Equivalent Conditions

There are several equivalent ways to state normality, each giving a different perspective. Let N ≤ G. The following are all equivalent:

(i)
gNg⁻¹ = N for all g ∈ G  (conjugation invariance)
(ii)
gN = Ng for all g ∈ G  (left cosets = right cosets)
(iii)
Every left coset of N in G is also a right coset
(iv)
The coset multiplication (aN)(bN) = (ab)N is well-defined
(v)
N is the kernel of some group homomorphism φ : G → H
(vi)
N is a union of conjugacy classes of G
Proof that (i) ⟺ (ii):

(i) ⟹ (ii): Suppose gNg⁻¹ = N for all g ∈ G. We want to show gN = Ng.
Take any x ∈ gN, so x = gn for some n ∈ N. Then:
x = gn = g·n·g⁻¹·g = (gng⁻¹)·g ∈ Ng (since gng⁻¹ ∈ N by hypothesis).
So gN ⊆ Ng. Applying the same to g⁻¹ gives g⁻¹N ⊆ Ng⁻¹, i.e. Ng ⊆ gN. Hence gN = Ng.

(ii) ⟹ (i): Suppose gN = Ng for all g ∈ G. Then gNg⁻¹ = (gN)g⁻¹ = (Ng)g⁻¹ = N(gg⁻¹) = N. ∎

Condition (vi): Union of Conjugacy Classes

The conjugacy class of x in G is Cl(x) = {gxg⁻¹ : g ∈ G}. The condition gNg⁻¹ = N for all g means N is closed under all conjugations — so if it contains one element of a conjugacy class, it contains all of them. This gives the elegant reformulation: N ◁ G if and only if N is a union of conjugacy classes of G.

§ 4

Examples and Non-Examples

Always-Normal Subgroups

Example 1 — Trivial Cases

For any group G, both {e} ◁ G and G ◁ G. These are called the trivial normal subgroups. Verify: g·e·g⁻¹ = e ∈ {e}, and gGg⁻¹ = G.

Example 2 — Abelian Groups

In an abelian group G, every subgroup is normal. Proof: gng⁻¹ = gg⁻¹n = n ∈ N for all g, n. So in ℤ, (ℚ, +), any cyclic group — all subgroups are normal.

Example 3 — The Alternating Group Aₙ in Sₙ

The alternating group Aₙ (even permutations) is a normal subgroup of Sₙ (all permutations) for all n ≥ 1. It has index 2, and any subgroup of index 2 is automatically normal. Proof: if [G:H] = 2, then G has only two left cosets {H, gH} and two right cosets {H, Hg}, so gH = Hg for all g.

Example 4 — The Center Z(G)

The center Z(G) = {g ∈ G : gx = xg for all x ∈ G} is always normal in G. Indeed, if z ∈ Z(G), then gzg⁻¹ = gg⁻¹z = z ∈ Z(G).

Example 5 — SL(n,ℝ) in GL(n,ℝ)

The special linear group SL(n,ℝ) = {A : det(A) = 1} is normal in GL(n,ℝ) = {A : det(A) ≠ 0}. This is because SL(n,ℝ) = ker(det), the kernel of the determinant homomorphism det : GL(n,ℝ) → ℝ*.

A Non-Example

Non-Example — H = {e, (12)} in S₃:
Let G = S₃ and H = {e, (12)}. Is H ◁ S₃? Take g = (13) ∈ S₃.
Then (13)(12)(13)⁻¹ = (13)(12)(13) = (23).
Since (23) ∉ H, we have gHg⁻¹ ≠ H. So H is not normal in S₃.

§ 5

Key Properties

Intersections and Products

Theorem

If N ◁ G and M ◁ G, then:

  1. N ∩ M ◁ G
  2. NM = {nm : n ∈ N, m ∈ M} ◁ G
Let x ∈ N ∩ M and g ∈ G. Then gxg⁻¹ ∈ N (since N ◁ G and x ∈ N) and gxg⁻¹ ∈ M (since M ◁ G and x ∈ M). Therefore gxg⁻¹ ∈ N ∩ M, so N ∩ M ◁ G. ∎

Normal Subgroups and Homomorphisms

Theorem

If φ : G → H is a group homomorphism, then:

  • ker(φ) ◁ G — the kernel is always normal
  • If N ◁ G, then φ(N) ◁ φ(G) (image of normal is normal in image)
  • If M ◁ H, then φ⁻¹(M) ◁ G (preimage of normal is normal)
Let n ∈ ker(φ), so φ(n) = e_H. For any g ∈ G:

φ(gng⁻¹) = φ(g)·φ(n)·φ(g⁻¹) = φ(g)·e_H·φ(g)⁻¹ = e_H.

So gng⁻¹ ∈ ker(φ). Since g and n were arbitrary, ker(φ) ◁ G. ∎

Normal Is Not Transitive

Caution: Normality is not transitive. If K ◁ H and H ◁ G, it does not follow that K ◁ G.

Counterexample: In G = D₄ (dihedral group of order 8), let H = ⟨r²,s⟩ ≅ ℤ₂×ℤ₂ and K = ⟨s⟩. Then K ◁ H and H ◁ G, but K is not normal in G.

Index-2 Subgroups

Theorem

Any subgroup of index 2 is normal.

This is one of the most useful normality criteria in practice. Proof: if [G:H] = 2, the only cosets of H are H itself and G\H. So the left coset gH equals H if g ∈ H, or G\H if g ∉ H — and the right coset Hg does the same. Hence gH = Hg for all g.

§ 6

Quotient Groups G/N

Here is the central payoff. Given N ◁ G, we define the quotient group (or factor group) G/N.

Definition — Quotient Group

Let N ◁ G. The set of all left cosets of N in G, denoted G/N = {gN : g ∈ G}, forms a group under the operation:

(aN)(bN) := (ab)N

The identity element is N (= eN), and the inverse of gN is g⁻¹N.

Well-Definedness

The critical step is verifying the operation is well-defined. Suppose aN = a′N and bN = b′N (same cosets, different representatives). We need (ab)N = (a′b′)N.

Since a′ ∈ aN, write a′ = an₁ for some n₁ ∈ N Since b′ ∈ bN, write b′ = bn₂ for some n₂ ∈ N Then: a′b′ = an₁·bn₂ = ab·(b⁻¹n₁b)·n₂ Since N ◁ G: b⁻¹n₁b ∈ N, so (b⁻¹n₁b)·n₂ ∈ N Therefore a′b′ = ab·(something in N), so (a′b′)N = (ab)N ✓

The Order of G/N

Lagrange's Theorem — Corollary

If G is finite, then |G/N| = |G|/|N| = [G : N], the index of N in G.

Examples of Quotient Groups

Example — ℤ/nℤ

The integers ℤ with the subgroup nℤ = {…, −2n, −n, 0, n, 2n, …} (multiples of n). Since ℤ is abelian, nℤ ◁ ℤ. The quotient ℤ/nℤ has n cosets: 0+nℤ, 1+nℤ, …, (n−1)+nℤ — these are just the residue classes mod n! So ℤ/nℤ ≅ ℤₙ.

Example — ℝ/ℤ

The quotient ℝ/ℤ consists of all real numbers modulo integer equivalence. The coset x + ℤ is characterized by the fractional part of x. This group is isomorphic to the circle group S¹ = {e^{2πiθ} : θ ∈ ℝ} under complex multiplication.

Example — S₃/A₃

A₃ ◁ S₃ with index 2. The quotient S₃/A₃ has 2 elements: {A₃, (12)A₃}. Any group of order 2 is isomorphic to ℤ₂. So S₃/A₃ ≅ ℤ₂.

The Canonical Projection

There is always a natural surjective homomorphism called the canonical projection (or quotient map):

π : G → G/N, defined by π(g) = gN It is surjective, and ker(π) = N.

This gives a concrete realization of N as a kernel — confirming that N ◁ G if and only if N is the kernel of some homomorphism out of G.

§ 7

The Isomorphism Theorems

Normal subgroups are the key ingredient in the three fundamental isomorphism theorems, which are the workhorses of group theory.

First Isomorphism Theorem

If φ : G → H is a group homomorphism, then ker(φ) ◁ G and:

G / ker(φ) ≅ Im(φ)

The isomorphism is given by g·ker(φ) ↦ φ(g).

Define ψ : G/ker(φ) → Im(φ) by ψ(gK) = φ(g) where K = ker(φ).

Well-defined: If gK = g′K, then g′⁻¹g ∈ K, so φ(g′⁻¹g) = e, giving φ(g) = φ(g′).
Homomorphism: ψ((aK)(bK)) = ψ(abK) = φ(ab) = φ(a)φ(b) = ψ(aK)ψ(bK).
Injective: If ψ(gK) = e, then φ(g) = e, so g ∈ K, meaning gK = K = identity coset.
Surjective: For any φ(g) ∈ Im(φ), ψ(gK) = φ(g). ∎
Second Isomorphism Theorem (Diamond Theorem)

Let H ≤ G and N ◁ G. Then:

  • HN = {hn : h ∈ H, n ∈ N} is a subgroup of G
  • H ∩ N ◁ H
  • H/(H ∩ N) ≅ HN/N
Third Isomorphism Theorem

Let N ◁ G and K ◁ G with N ≤ K ≤ G. Then K/N ◁ G/N and:

(G/N) / (K/N) ≅ G/K

Informally: "canceling N from top and bottom." This is the group-theory analogue of (a/b)/(c/b) = a/c.

Simple Groups

Definition — Simple Group

A group G is called simple if its only normal subgroups are {e} and G itself. Simple groups are the "atoms" of group theory — they cannot be further factored by quotients.

Examples of simple groups: ℤₚ for prime p, the alternating groups Aₙ for n ≥ 5. The classification of all finite simple groups is one of the greatest achievements of 20th-century mathematics, completed in 2004 across tens of thousands of pages.

Composition Series & The Jordan–Hölder Theorem

A composition series for G is a chain:

{e} = G₀ ◁ G₁ ◁ G₂ ◁ … ◁ Gₙ = G where each Gᵢ ◁ Gᵢ₊₁ and each quotient Gᵢ₊₁/Gᵢ is simple.
Jordan–Hölder Theorem

If a finite group G has a composition series, then any two composition series of G have the same length and the same multiset of composition factors (up to isomorphism and reordering).

This is the group-theoretic analogue of the Fundamental Theorem of Arithmetic: just as every integer factors uniquely into primes, every finite group has a unique factorization into simple groups.
