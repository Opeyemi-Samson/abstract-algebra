The Isomorphism Theorems

Three theorems that govern how groups relate to their quotients

Overview First Theorem Second Theorem Third Theorem Correspondence Applications Quiz
§ 1

Overview

The isomorphism theorems are the backbone of group theory. They describe, precisely and completely, how homomorphisms, subgroups, normal subgroups, and quotient groups interact. Once understood, they become automatic tools: whenever you see a homomorphism or a quotient, these theorems tell you exactly what structure to expect.

There are three main theorems (sometimes a fourth "correspondence theorem" is listed). They all flow from one central idea: a homomorphism from G decomposes G into cosets of its kernel, and those cosets form the quotient group.

Historical note: These theorems are due to Emmy Noether (1882–1935), who reformulated and vastly generalised earlier work of Jordan and Hölder. The theorems hold not just for groups but for rings, modules, and many other algebraic structures.
§ 2

1st First Isomorphism Theorem

First Isomorphism Theorem

Let φ : G → H be a group homomorphism. Then:

  1. ker(φ) ◁ G
  2. Im(φ) ≤ H
  3. G/ker(φ) ≅ Im(φ)

The isomorphism is given by the map ψ : g·ker(φ) ↦ φ(g).

G ──π──→ G/ker(φ) │ ↓ ψ ≅ └──φ────→ Im(φ) ≤ H φ = ψ ∘ π (factorisation through the quotient)

Every homomorphism factors as: a surjection onto a quotient, followed by an isomorphism, followed by an inclusion. This is the canonical factorisation of any group map.

Let K = ker(φ). We already know K ◁ G (proved in the homomorphisms note).

Define ψ : G/K → Im(φ) by ψ(gK) = φ(g).

Well-defined: If gK = g'K, then g'⁻¹g ∈ K = ker(φ), so φ(g'⁻¹g) = e, giving φ(g) = φ(g'). ✓

Homomorphism: ψ((aK)(bK)) = ψ(abK) = φ(ab) = φ(a)φ(b) = ψ(aK)ψ(bK). ✓

Injective: If ψ(gK) = e_H, then φ(g) = e_H, so g ∈ K, meaning gK = K (the identity of G/K). Thus ker(ψ) = {K}, so ψ is injective. ✓

Surjective onto Im(φ): For any φ(g) ∈ Im(φ), ψ(gK) = φ(g). ✓

Therefore ψ is an isomorphism G/K → Im(φ). ∎

How to Use It

The standard application: to identify a quotient group G/N, find a surjective homomorphism φ : G → H with kernel exactly N. Then G/N ≅ H.

Example 1 — ℤ/nℤ ≅ ℤₙ

φ : ℤ → ℤₙ, φ(k) = k mod n. This is surjective, and ker(φ) = nℤ. By the First Isomorphism Theorem: ℤ/nℤ ≅ ℤₙ.

Example 2 — GL(n)/SL(n) ≅ ℝ*

det : GL(n,ℝ) → ℝ* is surjective with kernel SL(n,ℝ). So GL(n,ℝ)/SL(n,ℝ) ≅ ℝ*.

Example 3 — ℝ/ℤ ≅ S¹

φ : ℝ → S¹, φ(x) = e^{2πix} is surjective with kernel ℤ. So ℝ/ℤ ≅ S¹.

2nd Second Isomorphism Theorem

Also known as the Diamond Theorem or Modular Law, due to the shape of the subgroup lattice it describes.

Second Isomorphism Theorem

Let H ≤ G and N ◁ G. Then:

  1. HN = {hn : h ∈ H, n ∈ N} is a subgroup of G
  2. H ∩ N ◁ H
  3. H / (H ∩ N) ≅ HN / N
Step 1 — HN ≤ G: Since N ◁ G, for any h ∈ H we have hNh⁻¹ = N, so HN = NH. Now: (h₁n₁)(h₂n₂) = h₁h₂(h₂⁻¹n₁h₂)n₂ ∈ HN (since h₂⁻¹n₁h₂ ∈ N). Also (hn)⁻¹ = n⁻¹h⁻¹ = h⁻¹(hn⁻¹h⁻¹) ∈ HN. ✓

Step 2 — H ∩ N ◁ H: For h ∈ H, x ∈ H ∩ N: hxh⁻¹ ∈ H (since H is a group) and hxh⁻¹ ∈ N (since N ◁ G). So hxh⁻¹ ∈ H ∩ N. ✓

Step 3 — Isomorphism: Define φ : H → HN/N by φ(h) = hN. This is a well-defined homomorphism and is surjective (every coset hnN = hN for h ∈ H). Its kernel is {h ∈ H : hN = N} = {h ∈ H : h ∈ N} = H ∩ N. By the First Isomorphism Theorem: H/(H ∩ N) ≅ HN/N. ∎
Example — In ℤ, H = 3ℤ, N = 5ℤ

HN = 3ℤ + 5ℤ = ℤ (since gcd(3,5) = 1). H ∩ N = 15ℤ. The theorem gives: 3ℤ/15ℤ ≅ ℤ/5ℤ = ℤ₅. Check: 3ℤ/15ℤ has order 5. ✓

§ 4

3rd Third Isomorphism Theorem

The Third Isomorphism Theorem is the group-theoretic analogue of cancelling fractions: (G/N)/(K/N) ≅ G/K.

Third Isomorphism Theorem

Let N ◁ G and K ◁ G with N ≤ K ≤ G. Then:

  1. K/N ◁ G/N
  2. (G/N) / (K/N) ≅ G/K
Analogy with fractions: (G/N)/(K/N) "=" G·N⁻¹ · (K·N⁻¹)⁻¹ = G·K⁻¹ = G/K The isomorphism sends: (gN)·(K/N) ↦ gK
Define φ : G/N → G/K by φ(gN) = gK.

Well-defined: If gN = g'N then g'⁻¹g ∈ N ≤ K, so gK = g'K. ✓

Homomorphism: φ(gN · g'N) = φ(gg'N) = gg'K = gK · g'K = φ(gN)φ(g'N). ✓

Surjective: For any gK ∈ G/K, φ(gN) = gK. ✓

Kernel: ker(φ) = {gN : gK = K} = {gN : g ∈ K} = K/N. ✓

By the First Isomorphism Theorem: (G/N)/ker(φ) ≅ Im(φ), i.e. (G/N)/(K/N) ≅ G/K. ∎
Example — Quotients of ℤ

N = 12ℤ ≤ K = 4ℤ ≤ G = ℤ. Then (ℤ/12ℤ)/(4ℤ/12ℤ) ≅ ℤ/4ℤ. Here 4ℤ/12ℤ is the subgroup {0, 4, 8} of ℤ₁₂, i.e. the subgroup of order 3 inside ℤ₁₂. And ℤ₁₂ / ℤ₃ ≅ ℤ₄. ✓

§ 5

The Correspondence Theorem

Sometimes listed as the Fourth Isomorphism Theorem, this theorem describes the complete subgroup structure of a quotient group.

Correspondence (Lattice) Theorem

Let N ◁ G and π : G → G/N be the canonical projection. There is an order-preserving bijection between:

  • The set of subgroups of G containing N, and
  • The set of subgroups of G/N

given by H ↦ H/N = π(H) and its inverse L ↦ π⁻¹(L).

Moreover: H ◁ G iff H/N ◁ G/N, and [G:H] = [G/N : H/N].

Example — Subgroup Lattice of ℤ/12ℤ

Subgroups of ℤ containing 12ℤ: 12ℤ, 6ℤ, 4ℤ, 3ℤ, 2ℤ, ℤ (those nℤ with n | 12). These correspond to subgroups of ℤ₁₂ of orders 1, 2, 3, 4, 6, 12 respectively.

§ 6

Applications and Strategy

The Standard Template

To prove G/N ≅ H using the First Isomorphism Theorem:

  1. Construct a homomorphism φ : G → H
  2. Prove φ is surjective
  3. Compute ker(φ) and show ker(φ) = N
  4. Conclude G/N ≅ H by the First Isomorphism Theorem
Application — ℝ*/ℝ₊* ≅ ℤ₂

φ : ℝ* → {±1} = ℤ₂, φ(x) = x/|x| (the sign function). Surjective, ker = ℝ₊*. So ℝ*/ℝ₊* ≅ ℤ₂.

Application — Cyclic Groups

Every cyclic group of order n is isomorphic to ℤₙ. Proof: let G = ⟨g⟩. Define φ : ℤ → G by φ(k) = gᵏ. This is a surjective homomorphism with kernel nℤ (since gⁿ = e and no smaller power works). By the First Isomorphism Theorem: G ≅ ℤ/nℤ = ℤₙ.

Application — Product Groups

Consider G = ℤ₁₂ and N = ⟨4⟩ = {0,4,8} ≅ ℤ₃. Then G/N ≅ ℤ₄. Now take K = ⟨6⟩ = {0,6} ≅ ℤ₂. Then G/K ≅ ℤ₆. The Third Isomorphism Theorem: if we take N ≤ K, it tells us the relationship between these quotients.
