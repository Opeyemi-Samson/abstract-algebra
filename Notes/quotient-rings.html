Factor rings, ring homomorphisms, and the isomorphism theorems for rings

§ 1

Motivation

Just as we formed quotient groups by collapsing a normal subgroup to the identity, we can form quotient rings by collapsing an ideal to zero. The idea is to work "modulo I" — declaring that two ring elements are the same if their difference lies in I.

The most familiar example is modular arithmetic: in ℤ/nℤ, we declare a ≡ b whenever n | (a − b). The result is a new ring where n = 0. This is the simplest case of a quotient ring.

Why ideals (not just subrings)? To make the quotient R/I into a ring, we need the coset multiplication (a+I)(b+I) = ab+I to be well-defined. This works if and only if I is an ideal.
§ 2

Constructing the Quotient Ring

Cosets and the Underlying Set

Let R be a ring and I an ideal. As an abelian group, (I, +) ◁ (R, +), so we can form the quotient group R/I. Its elements are cosets:

R/I = { a + I : a ∈ R } = { a + I : a ∈ R } where a + I = { a + x : x ∈ I }

Two elements a, b ∈ R give the same coset iff a − b ∈ I. We write a ≡ b (mod I).

The Ring Operations

Definition — Quotient Ring

Let I be an ideal of R. The quotient ring R/I has underlying set {a+I : a ∈ R} with operations:

Addition: (a + I) + (b + I) = (a + b) + I Multiplication: (a + I)(b + I) = (ab) + I

Well-Definedness of Multiplication

Addition is well-defined since I is a subgroup. For multiplication: suppose a+I = a'+I and b+I = b'+I, so a' = a+i and b' = b+j for i,j ∈ I. Then:

a'b' = (a+i)(b+j) = ab + aj + ib + ij Since I is an ideal: aj ∈ I, ib ∈ I, ij ∈ I So a'b' − ab = aj + ib + ij ∈ I, giving a'b' + I = ab + I ✓
Theorem

R/I with these operations is a ring. Its zero element is 0+I = I, and if R has unity 1, then R/I has unity 1+I.

The Canonical Projection

The map π : R → R/I, π(a) = a+I is a surjective ring homomorphism with kernel I. This is the ring analogue of the group quotient map.

§ 3

Ring Homomorphisms

Definition — Ring Homomorphism

A function φ : R → S between rings is a ring homomorphism if for all a, b ∈ R:

φ(a + b) = φ(a) + φ(b) (preserves addition) φ(ab) = φ(a)φ(b) (preserves multiplication) φ(1_R) = 1_S (preserves unity, if both rings have unity)

The kernel of a ring homomorphism is ker(φ) = {r ∈ R : φ(r) = 0}. Unlike in group theory, the kernel of a ring homomorphism is always an ideal (not just a normal subgroup).

Theorem

For a ring homomorphism φ : R → S:

  • ker(φ) is an ideal of R
  • Im(φ) is a subring of S
  • φ is injective iff ker(φ) = {0}
Examples of Ring Homomorphisms
  • Evaluation: ev_a : R[x] → R, ev_a(f) = f(a). ker = {f : f(a) = 0} = ⟨x−a⟩ if R is a field.
  • Reduction mod n: φ : ℤ → ℤₙ, φ(k) = k mod n. ker = nℤ.
  • Inclusion: ℤ → ℚ, n ↦ n. Injective, ker = {0}.
  • Complex conjugation: ℂ → ℂ, a+bi ↦ a−bi. Bijective ring homomorphism (automorphism).
§ 4

Isomorphism Theorems for Rings

The isomorphism theorems carry over from groups to rings almost word-for-word, replacing "normal subgroup" with "ideal" and "group homomorphism" with "ring homomorphism".

First Isomorphism Theorem for Rings

Let φ : R → S be a ring homomorphism. Then ker(φ) is an ideal of R and:

R / ker(φ) ≅ Im(φ)

via the isomorphism ψ : a + ker(φ) ↦ φ(a).

Define ψ(a + K) = φ(a) where K = ker(φ).

Well-defined: If a+K = b+K then a−b ∈ K so φ(a−b)=0, giving φ(a)=φ(b). ✓

Ring homomorphism:
ψ((a+K)+(b+K)) = ψ((a+b)+K) = φ(a+b) = φ(a)+φ(b) = ψ(a+K)+ψ(b+K). ✓
ψ((a+K)(b+K)) = ψ(ab+K) = φ(ab) = φ(a)φ(b) = ψ(a+K)ψ(b+K). ✓

Injective: ψ(a+K) = 0 ⟹ φ(a) = 0 ⟹ a ∈ K ⟹ a+K = K = 0_{R/K}. ✓

Surjective onto Im(φ): For φ(a) ∈ Im(φ), ψ(a+K) = φ(a). ✓   ∎
Second Isomorphism Theorem for Rings

Let S be a subring and I an ideal of R. Then S+I is a subring of R, S∩I is an ideal of S, and:

S / (S ∩ I) ≅ (S + I) / I
Third Isomorphism Theorem for Rings

Let I ⊆ J be ideals of R. Then J/I is an ideal of R/I and:

(R/I) / (J/I) ≅ R/J
Correspondence Theorem for Rings

There is an order-preserving bijection between ideals of R/I and ideals of R containing I.

§ 5

Key Examples

Example 1 — ℤ/nℤ = ℤₙ

The quintessential example. R = ℤ, I = nℤ. The coset a + nℤ is the residue class of a mod n. The ring operations are addition and multiplication mod n. ℤ/nℤ is a field iff n is prime (since ⟨n⟩ is maximal in ℤ iff n is prime).

Example 2 — ℝ[x]/⟨x²+1⟩ ≅ ℂ

This is one of the most beautiful examples in algebra. Consider R = ℝ[x] and I = ⟨x²+1⟩. Every coset has a unique representative of the form a + bx (reduce mod x²+1). The coset x+I satisfies (x+I)² = x²+I = (−1)+I. So x+I behaves exactly like i = √−1. The map:

φ : ℝ[x]/⟨x²+1⟩ → ℂ a + bx + ⟨x²+1⟩ ↦ a + bi

is a ring isomorphism. So the complex numbers arise naturally as a quotient ring of real polynomials!

Example 3 — ℝ[x]/⟨x²−1⟩

Now I = ⟨x²−1⟩ = ⟨(x−1)(x+1)⟩. Since x²−1 = (x−1)(x+1) is reducible, ⟨x²−1⟩ is not prime, so R/I is not an integral domain. Indeed, (x−1+I)(x+1+I) = x²−1+I = 0+I, giving zero divisors. By CRT: ℝ[x]/⟨x²−1⟩ ≅ ℝ × ℝ.

Example 4 — F[x]/⟨p(x)⟩ for Irreducible p

Let F be a field and p(x) ∈ F[x] irreducible. Since F[x] is a PID and p is irreducible, ⟨p(x)⟩ is maximal. Therefore F[x]/⟨p(x)⟩ is a field. This is the standard construction of field extensions — adjoining a root of p to F.

Example: ℚ[x]/⟨x²−2⟩ ≅ ℚ(√2) = {a + b√2 : a,b ∈ ℚ}.

Example 5 — Dual Numbers: ℝ[x]/⟨x²⟩

R = ℝ[x], I = ⟨x²⟩. Every coset is a + bx + ⟨x²⟩ with (bx)² = 0. The element ε = x+I satisfies ε² = 0 but ε ≠ 0. The ring ℝ[ε] = {a + bε : a,b ∈ ℝ, ε² = 0} is called the ring of dual numbers, used in automatic differentiation.
