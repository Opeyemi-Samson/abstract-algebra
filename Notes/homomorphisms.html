Group Homomorphisms

Structure-preserving maps between groups

Definition Properties Types Kernel & Image Examples Automorphisms Quiz
§ 1

Definition

A group homomorphism is a map between two groups that respects the group structure — it interacts compatibly with the group operations. This makes it the natural notion of "map" in group theory, just as continuous maps are the natural maps in topology or linear maps in linear algebra.

Definition — Group Homomorphism

Let (G, ·) and (H, ★) be groups. A function φ : G → H is a group homomorphism if for all a, b ∈ G:

φ(a · b) = φ(a) ★ φ(b)

In words: the image of a product equals the product of the images. The map φ does not need to be bijective — it just needs to preserve the operation.

Intuition: A homomorphism is a "shadow" of G in H — it may collapse some structure (several elements might map to the same image), but whatever structure survives in the image is an accurate reflection of G's structure.
§ 2

Basic Properties

Theorem — Elementary Properties

Let φ : G → H be a group homomorphism. Then:

  1. φ(e_G) = e_H   (identity maps to identity)
  2. φ(g⁻¹) = φ(g)⁻¹   (inverse maps to inverse)
  3. φ(gⁿ) = φ(g)ⁿ   for all n ∈ ℤ
  4. If |g| is finite, then |φ(g)| divides |g|
  5. If K ≤ G, then φ(K) ≤ H   (subgroups map to subgroups)
  6. If L ≤ H, then φ⁻¹(L) ≤ G   (preimages of subgroups are subgroups)
(1): φ(e_G) = φ(e_G · e_G) = φ(e_G) · φ(e_G). Cancelling φ(e_G) on the left gives e_H = φ(e_G). ∎

(2): φ(g) · φ(g⁻¹) = φ(g · g⁻¹) = φ(e_G) = e_H. So φ(g⁻¹) is the inverse of φ(g), i.e. φ(g⁻¹) = φ(g)⁻¹. ∎

(4): Let n = |g|, so gⁿ = e. Then φ(g)ⁿ = φ(gⁿ) = φ(e) = e. So the order of φ(g) divides n = |g|.

Composition of Homomorphisms

If φ : G → H and ψ : H → K are homomorphisms, then ψ ∘ φ : G → K is also a homomorphism:

(ψ ∘ φ)(ab) = ψ(φ(ab)) = ψ(φ(a)φ(b)) = ψ(φ(a))ψ(φ(b)) = (ψ∘φ)(a)(ψ∘φ)(b)

This means groups and homomorphisms form a category, denoted Grp.

Theorem — Injective ↔ Trivial Kernel

A homomorphism φ : G → H is injective if and only if ker(φ) = {e_G}.

(⟹): If φ is injective and g ∈ ker(φ), then φ(g) = e_H = φ(e_G), so g = e_G. Thus ker(φ) = {e_G}.

(⟸): Suppose ker(φ) = {e_G}. If φ(a) = φ(b), then φ(ab⁻¹) = φ(a)φ(b)⁻¹ = e_H, so ab⁻¹ ∈ ker(φ) = {e_G}, giving a = b. ∎
§ 4

Kernel and Image

Definition — Kernel and Image

For φ : G → H:

ker(φ) = { g ∈ G : φ(g) = e_H } ≤ G Im(φ) = { φ(g) : g ∈ G } ≤ H
Theorem

For any homomorphism φ : G → H:

  • ker(φ) is a normal subgroup of G
  • Im(φ) is a subgroup of H (not necessarily normal)
  • For any a ∈ G, the fiber φ⁻¹({φ(a)}) = a·ker(φ) — the fiber over any point is a coset of the kernel
We already know ker(φ) ≤ G. To show normality, take g ∈ G, n ∈ ker(φ):

φ(gng⁻¹) = φ(g)·φ(n)·φ(g⁻¹) = φ(g)·e_H·φ(g)⁻¹ = e_H.

So gng⁻¹ ∈ ker(φ), proving ker(φ) ◁ G. ∎

Fibers are Cosets

A crucial structural insight: all elements mapping to the same value form a coset of the kernel. If φ(a) = h, then:

φ⁻¹({h}) = { g ∈ G : φ(g) = h } = { g ∈ G : φ(g) = φ(a) } = { g ∈ G : φ(a⁻¹g) = e } = { g ∈ G : a⁻¹g ∈ ker(φ) } = a · ker(φ)

This is why the First Isomorphism Theorem works: G/ker(φ) literally consists of these fibers, and they are in bijection with Im(φ).

§ 5

Important Examples

Example 1 — Determinant

φ : GL(n, ℝ) → (ℝ*, ×), φ(A) = det(A). This is a surjective homomorphism because det(AB) = det(A)det(B). The kernel is SL(n, ℝ) = {A : det(A) = 1}. So SL(n, ℝ) ◁ GL(n, ℝ) and GL(n,ℝ)/SL(n,ℝ) ≅ ℝ*.

Example 2 — Sign of a Permutation

sgn : Sₙ → {±1} ≅ ℤ₂, sgn(σ) = +1 if σ is even, −1 if odd. This is a surjective homomorphism and ker(sgn) = Aₙ (the alternating group). So Aₙ ◁ Sₙ and Sₙ/Aₙ ≅ ℤ₂.

Example 3 — Exponential Map

φ : (ℝ, +) → (ℝ₊*, ×), φ(x) = eˣ. This is an isomorphism: e^{x+y} = eˣ·eʸ, and it is bijective. So (ℝ, +) ≅ (ℝ₊*, ×) as groups.

Example 4 — Modular Reduction

φ : ℤ → ℤₙ, φ(k) = k mod n. This is a surjective homomorphism with kernel nℤ. By the First Isomorphism Theorem: ℤ/nℤ ≅ ℤₙ.

Example 5 — Trivial and Identity Homomorphisms

For any groups G, H: the trivial homomorphism φ(g) = e_H for all g, with ker = G. The identity id_G : G → G, id(g) = g, is an automorphism with ker = {e}.

Example 6 — Cayley's Theorem

Every group G embeds (injects) into a symmetric group. The map φ : G → Sym(G) defined by φ(g) = (left multiplication by g) is an injective homomorphism. This means every abstract group is concretely realised as a group of permutations.

§ 6

Automorphisms

An automorphism of G is a bijective homomorphism G → G — a symmetry of the group itself. The set of all automorphisms forms a group under composition.

Definition — Automorphism Group

Aut(G) = {φ : G → G | φ is an automorphism}, with composition as the group operation.

Inner Automorphisms

For each g ∈ G, conjugation by g defines an automorphism:

φ_g : G → G, φ_g(x) = gxg⁻¹ φ_g is a bijective homomorphism (its inverse is φ_{g⁻¹}).

Automorphisms of this form are called inner automorphisms. The set of all inner automorphisms is:

Inn(G) = { φ_g : g ∈ G } ◁ Aut(G)
Theorem

Inn(G) ◁ Aut(G), and there is an isomorphism Inn(G) ≅ G/Z(G), where Z(G) is the center of G.

The outer automorphism group is Out(G) = Aut(G)/Inn(G). Outer automorphisms are the "exotic" symmetries of G that cannot be realised by conjugation.

Example — Aut(ℤₙ) ≅ (ℤₙ)*

Every automorphism of ℤₙ = ⟨1⟩ is determined by where 1 goes. We need φ(1) to generate ℤₙ, so φ(1) = k where gcd(k, n) = 1. Therefore Aut(ℤₙ) ≅ (ℤₙ)* (the group of units mod n), which has order φ(n) (Euler's totient).

Example — Aut(S₃) ≅ S₃

S₃ has trivial center, so Inn(S₃) ≅ S₃/Z(S₃) = S₃/{e} ≅ S₃. One can check all automorphisms of S₃ are inner, so Aut(S₃) ≅ S₃ (S₃ is a "complete" group).
