Building new groups from old ones by collapsing structure

§ 1

Motivation — Collapsing Structure

One of the most powerful ideas in algebra is identifying things together. Given a group G and a normal subgroup N, we want to build a new, simpler group by treating every element of N as "equivalent to the identity." All elements in the same coset of N get identified as a single element.

Think of an analogy: in arithmetic mod 4, we identify 0, 4, 8, 12, … as all being "the same." The integers ℤ collapse to ℤ₄ = {0, 1, 2, 3}. Quotient groups are exactly this — a systematic, algebraically rigorous way to collapse a group by one of its normal subgroups.

Key idea: A quotient group G/N doesn't just forget N — it wraps G around N, so that N becomes the identity element and everything else is measured relative to N.
§ 2

The Construction

Step 1 — Cosets as Elements

Let N ◁ G. The elements of G/N are the left cosets of N:

G/N = { gN : g ∈ G } where gN = { gn : n ∈ N }

Two elements g, h ∈ G define the same coset (gN = hN) exactly when h⁻¹g ∈ N, i.e. when g and h differ by an element of N. So cosets partition G into equal-sized blocks.

Step 2 — The Group Operation

Definition — Quotient Group Operation

Define multiplication on G/N by: (aN)(bN) = (ab)N. This makes G/N into a group called the quotient group (or factor group) of G by N.

Step 3 — Verifying Well-Definedness

The operation must not depend on which representative we choose. Suppose aN = a'N and bN = b'N. Then a' = an₁ and b' = bn₂ for some n₁, n₂ ∈ N. We need (a'b')N = (ab)N:

a'b' = an₁·bn₂ = ab·(b⁻¹n₁b)·n₂ Since N ◁ G: b⁻¹n₁b ∈ N, so (b⁻¹n₁b)·n₂ ∈ N Therefore a'b' = ab·(element of N), so (a'b')N = (ab)N ✓

This is exactly where normality is essential — without it, b⁻¹n₁b might escape N and the operation would be undefined.

Step 4 — Group Axioms

Theorem

G/N with the operation (aN)(bN) = (ab)N is a group, where:

  • Identity: eN = N
  • Inverse: (gN)⁻¹ = g⁻¹N
  • Associativity: inherited from G
Identity: (gN)(eN) = (ge)N = gN. Similarly (eN)(gN) = gN. ✓

Inverse: (gN)(g⁻¹N) = (gg⁻¹)N = eN = N. ✓

Associativity: ((aN)(bN))(cN) = (ab)N·cN = ((ab)c)N = (a(bc))N = aN·(bc)N = (aN)((bN)(cN)). ✓ ∎

The Order Formula

Lagrange's Theorem — Corollary

If G is finite with N ◁ G, then |G/N| = |G| / |N|.

Each coset gN has exactly |N| elements (the map n ↦ gn is a bijection N → gN), and the cosets partition G. So there are exactly |G|/|N| cosets.

Worked Examples

Example 1 — ℤ/nℤ ≅ ℤₙ

Let G = ℤ (integers under addition) and N = nℤ = {…, -2n, -n, 0, n, 2n, …}. Since ℤ is abelian, nℤ ◁ ℤ. The cosets are the residue classes:

0 + nℤ = {…,-2n, -n, 0, n, 2n,…}

1 + nℤ = {…,1-2n, 1-n, 1, 1+n,…}

⋮ up to (n−1) + nℤ. The quotient ℤ/nℤ has exactly n elements, and its operation is addition mod n. So ℤ/nℤ ≅ ℤₙ.

Example 2 — S₃/A₃ ≅ ℤ₂

G = S₃ (order 6), N = A₃ = {e, (123), (132)} (order 3, index 2). The two cosets are:

A₃ = { e, (123), (132) } ← identity coset (12)A₃ = { (12), (23), (13) } ← the other coset

G/N = {A₃, (12)A₃} with multiplication: any two distinct cosets multiply to the other, giving a group of order 2. So S₃/A₃ ≅ ℤ₂.

Example 3 — ℝ/ℤ ≅ S¹

G = (ℝ, +), N = ℤ. The coset x + ℤ is determined by the fractional part {x} ∈ [0,1). The quotient ℝ/ℤ wraps the real line into a circle. The isomorphism ℝ/ℤ → S¹ is given by:

φ : ℝ/ℤ → S¹, x + ℤ ↦ e^{2πix}

This identifies each equivalence class with a point on the unit circle in ℂ. The group operation on S¹ is multiplication of complex numbers of modulus 1.

Example 4 — ℝ*/⟨-1⟩ ≅ ℝ₊*

Let G = (ℝ*, ×) and N = {1, -1} ≅ ℤ₂. The cosets are {r, -r} for each r > 0. The quotient identifies each nonzero real with its absolute value. The isomorphism ℝ*/⟨-1⟩ → ℝ₊* is (aN) ↦ |a|.

Example 5 — Dihedral: D₆/⟨r²⟩

The dihedral group D₆ = ⟨r, s | r⁶ = s² = e, srs = r⁻¹⟩ has order 12. The subgroup N = ⟨r²⟩ = {e, r², r⁴} is normal (index 4). The quotient D₆/N has order 4. One can verify it is isomorphic to the Klein four-group V₄ = ℤ₂ × ℤ₂.

§ 4

The Universal Property

The quotient group G/N satisfies a precise universal property that characterises it up to isomorphism. This is one of the deepest ways to understand what a quotient group really is.

Universal Property of Quotient Groups

Let N ◁ G and let π : G → G/N be the canonical projection π(g) = gN. If φ : G → H is any group homomorphism with N ⊆ ker(φ), then there exists a unique homomorphism φ̄ : G/N → H such that φ = φ̄ ∘ π.

G ──π──→ G/N │ ↘ φ̄ (unique) └──φ──→ H φ̄(gN) = φ(g) (well-defined since N ⊆ ker φ)

In plain language: any homomorphism out of G that kills N factors uniquely through G/N. This means G/N is the "most efficient" way to kill N — it kills exactly N and nothing more.

Existence: Define φ̄(gN) = φ(g). This is well-defined: if gN = g'N then g'⁻¹g ∈ N ⊆ ker φ, so φ(g'⁻¹g) = e, giving φ(g) = φ(g'). φ̄ is a homomorphism: φ̄((aN)(bN)) = φ̄(abN) = φ(ab) = φ(a)φ(b) = φ̄(aN)φ̄(bN).

Uniqueness: If ψ : G/N → H also satisfies ψ ∘ π = φ, then for any gN ∈ G/N: ψ(gN) = ψ(π(g)) = φ(g) = φ̄(gN). So ψ = φ̄. ∎

The Canonical Projection

The map π : G → G/N, g ↦ gN is always a surjective group homomorphism called the canonical projection or quotient map. Its kernel is exactly N:

ker(π) = { g ∈ G : gN = N } = { g ∈ G : g ∈ N } = N
§ 5

Structure Theorems

When is G/N Abelian?

Theorem

G/N is abelian if and only if N contains the commutator subgroup [G, G] = ⟨aba⁻¹b⁻¹ : a,b ∈ G ⟩.

The commutator subgroup [G,G] (also written G') is itself a normal subgroup of G, and G/[G,G] is the largest abelian quotient of G, called the abelianisation of G.

Example — Abelianisation of S₃

The commutator subgroup of S₃ is A₃ (generated by all 3-cycles). So the abelianisation is S₃/A₃ ≅ ℤ₂ — the largest abelian quotient of S₃.

Cyclic Quotients

Theorem

If G/Z(G) is cyclic (where Z(G) is the center), then G is abelian.

This is a very useful contrapositive: if G is non-abelian, then G/Z(G) cannot be cyclic. So for example, G/Z(G) ≅ ℤₙ is impossible for non-abelian G — the quotient by the center is never cyclic in the non-abelian case.

Correspondence Theorem

Correspondence (Lattice) Theorem

Let N ◁ G. There is a bijection between:

  • Subgroups of G/N, and
  • Subgroups of G that contain N

This bijection is H/N ↔ H. Moreover, H ◁ G iff H/N ◁ G/N, and [G:H] = [G/N : H/N].

The lattice theorem says: the subgroup structure of G/N is a perfect mirror of the subgroup structure of G "above" N. Every subgroup of the quotient lifts uniquely to a subgroup of G containing N.

Example — Subgroups of ℤ/12ℤ

The subgroups of ℤ/12ℤ correspond to subgroups of ℤ containing 12ℤ, i.e. the subgroups nℤ with n | 12: these are ℤ, 2ℤ, 3ℤ, 4ℤ, 6ℤ, 12ℤ. Correspondingly, ℤ₁₂ has subgroups of orders 1, 2, 3, 4, 6, 12.
