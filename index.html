About This Site

Info

A personal mathematics library

Each note is a fully self-contained HTML file — no frameworks, no build step, no internet required beyond fonts. Every note includes formal definitions, worked examples, proofs, and interactive elements to make the mathematics tangible and memorable.

To add a new note: drop an HTML file into the notes/ folder and add a card to index.html.
