## About

This repository contains a complete undergraduate-level Abstract Algebra course presented as simple, self-contained HTML pages. No frameworks, no JavaScript required.

**Topics covered:**
- Group Theory (groups, subgroups, homomorphisms, Sylow theorems, etc.)
- Ring Theory (rings, ideals, polynomial rings)
- Field Theory & Galois Theory (introduction)
